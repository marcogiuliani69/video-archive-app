# Prompt 01 — Project Setup

Read the following project documentation:

- docs/spec.md
- docs/api.md
- docs/architecture.md

This repository contains a web application for browsing and managing video files stored on a Ubuntu server.

The system architecture is:

Frontend

- React
- TypeScript
- Vite
- fetch API
- React Router

Backend

- Node.js
- Express
- TypeScript
- SQLite database

Runtime

- Ubuntu
- PM2 process manager

Your task is to prepare the initial project structure.

Requirements:

1. Propose a clean full-stack repository structure
2. Separate frontend and backend projects
3. Follow the architecture defined in docs/architecture.md
4. Use modular folder organization
5. Do not implement business logic yet
6. Focus only on project scaffolding

Generate:

- backend folder structure
- frontend folder structure
- main configuration files
- package.json files
- TypeScript configuration
- environment variables template (.env.example)

Output the proposed directory structure and explain briefly each major folder.
